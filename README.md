# SecretMessage

### About 
A python GUI to ```Encrypt & Decrypt``` a message with a ```SECRET KEY```.

### To use the GUI run the following file 
```
main.py
```

### Password
Use the password as ```1234```. You can change and keep your own password as well.


### Requirements 
* Tkinter
* pyBase64
